Every AI writing tool's marketing page promises the same five things. Here's how to cut through that and figure out in 30 minutes whether the tool actually works for you.

## The framework

We score four dimensions. You can do the same on a free trial.

### 1. The "boring task" test (10 minutes)

Pick a real piece of work you have to do this week. Run it through the tool exactly as you'd give the task to a human. **Don't** craft a clever prompt. The marketing pages are sold on clever-prompt outputs; you need to know how it behaves on tired, unedited inputs.

Grade: How much of the draft survives a normal editing pass?

### 2. The voice test (5 minutes)

Generate the same paragraph in three voices: casual, technical, and "your brand". Without naming the voices, ask someone else to guess which is which.

Grade: Can a stranger tell them apart?

### 3. The grounding test (10 minutes)

Attach a real source document. Generate output that should cite it. Spot-check three citations.

Grade: How many citations point to real things that actually say what was claimed?

### 4. The "second day" test (5 minutes)

Come back tomorrow. Open the tool fresh. How long does it take to remember how to do the thing you did yesterday?

Grade: Friction-free or fiddly?

## What scoring tells you

If a tool scores above 4 on tests 1 and 2 but fails 3, it's a drafting tool — good for blog posts, bad for documentation.

If it scores high on 3 but low on 1, it's a research tool that can write — useful in regulated contexts, painful for casual content.

If it fails 4, it doesn't matter how good the other scores are; you won't keep using it.

## What we don't grade on

- Sample galleries on the marketing site (cherry-picked)
- Hype-driven Twitter demos (cherry-picked)
- Total feature count (more features ≠ better)
- Model name (the same underlying model, wrapped in a worse product, is a worse product)
